Release Notes

HELIOS Parse / View 0.9.1, released 2026-03-20. Code developed by Dmitrii Bespalov at European XFEL.

Documentation Index User Guide Developer Guide README

Why 0.9.1

This is a patch release on top of the broader 0.9.0 documentation and bundle pass. The project is still pre-1.0 because several advanced scientific diagnostics remain intentionally quick-look rather than final forward models, but the shipping code and docs now better match each other.

Highlights in this release

  • user and developer docs refreshed again to match the current shipping Transmission/XCOM behavior
  • release metadata bumped to 0.9.1 with updated bundle readme and package version
  • release notes now include a concrete changelog section rather than only a high-level summary
  • shareable Windows/Linux source bundle rebuilt from the current code state

Changelog

0.9.1

  • documented model-aware Transmission with explicit mode selection and selected-result traces
  • documented the XCOM seam, canonical material resolution from parsed EOS/opacity metadata, and precomputed cold-XCOM fallback-table behavior
  • documented expanded plasmon/XRTS quick-look modes and warning semantics for degenerate, non-collective, or out-of-domain states
  • updated bundle-facing quick-start text and release metadata
  • rebuilt the release archive from the current repository state

0.9.0

  • major documentation expansion for users, developers, maintainers, and collaborators
  • release metadata and attribution added to About text, README, docs, and package metadata
  • shareable Windows/Linux source bundle with launch/bootstrap scripts and included demo HDF5 runs
  • current WaveFront, interface-event, and Preheat behavior documented with screenshots and interpretation notes

Bundle contents

  • source tree under src/
  • updated documentation under docs/
  • tests and supporting scripts
  • Windows launcher: Run_HELIOS_Analyzer.ps1
  • Linux launcher: run_helios_analyzer.sh
  • top-level quick start: README_FIRST.md
  • selected demo HDF5 examples under examples/

The 0.9.1 bundle also carries the current Transmission/XCOM code and tests, including the precomputed cold-material fallback-table path.
